# Vocaber

Vocaber provides a way of checking a word definition in the middle of a read and learn from it later.
The application is designed to work with french words and [Larousse dictionnary](https://www.larousse.fr/) but feel free to change the **fetchDefinition** function to adapt it to your language.

> :warning: Every word entered in the application will be stored locally on your device with its definition, there is no backend interaction so no security risk but no backup either.
